NSC23: The Dimensions

ส่งเข้าประกวดรายการ โครงการแข่งขันพัฒนาโปรแกรมคอมพิวเตอร์แห่งประเทศไทย ครั้งที่ 23 ประจำปี 2564 (NSC23)
ประเภท: โปรแกรมเพื่อความบันเทิง ในชื่อ วิบัติกาลเวลา (The dimension) เป็นบอร์ดเกมที่สร้างโดยอุปกรณ์ฮาร์ดแวร์

วัตถุประสงค์

  • นำเทคโนโลยีในปัจจุบันมาประยุกตใช้ในด้านบอร์ดเกม เพื่อพัฒนาวงการบอร์ดเกมมิให้ความแปลกใหม่และหลากหลายมากขึ้น
  • เพื่อนำความรู้ด้านซอฟต์แวร์และฮาร์ดแวร์มาประยกต์ใช้ในบอร์ดเกม
  • เพื่อสร้างเกมให้ความบันเทิง
  • เพื่อทดลองสร้างเกมในรูปแบบอนาล็อก จาก Arduino

    • อุปกรณ์ฮาร์ดแวร์

  • Arduino Mega 2560
  • TCA9548A I2C
  • OLED I2C
  • BUTTON SWITCH MODULE
  • LED
  • RGB MODULE
  • BETTERY MODULE

    • สร้างโดยโปรแกรม Arduino pro ide
